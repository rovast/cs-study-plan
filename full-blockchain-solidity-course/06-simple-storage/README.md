# ethers simple storage fcc

使用 ethers(5.x) + Ganache 部署和调用合约。

1. 使用本地启动的区块链服务
2. 调用合约的相关方法

## Install

```bash
pnpm install ethers@5 --registry=https://registry.npmmirror.com 
```

## Compile contract

```bash
pnpm compile
```

## Run

```bash
node deploy.js
```
